The Keboola integration uses OpenLineage to collect and send metadata to Grai.
Complete documentation for integrating OpenLineage with Keboola can be found [here](https://github.com/keboola/wr-openlineage).

### Webhook

<Callout type="info" emoji="ℹ️">
  In order to authenticate your webhook, you will need to create an API key in
  the Grai web app.
</Callout>

Grai will open an endpoint at `/api/v1/openlineage/<connection_id>/` which can be used to receive OpenLineage events.

See the [OpenLineage docs](https://github.com/keboola/wr-openlineage) for how to configure your client to send OpenLineage events.
As part of the configuration settings you'll need to provide component configuration and will need the variables below.

```bash copy
openlineage_api_url=https://api.grai.io/api/v1/openlineage/<connection_id>
```
