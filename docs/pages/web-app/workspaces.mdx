# Workspaces

Grai server can work with multiple workspaces, for example you can have a demo workspace for trying things out.

## Change workspace

1. Open the profile menu from the main app drawer and then choose Workspaces.

<br />
<Image
  src="/images/docs/core/webapp/change-workspace.png"
  height={600}
  width={400}
  alt="Change workspace page"
/>

2. Select a workspace from the list.

<Callout type="info" emoji="ℹ️">
  If you can't see a workspace, you may need to ask your administrator for an
  invitation.
</Callout>

## Add a workspace

Each workspace has its own set of users, connections, and nodes and edges.

1. Open the profile menu from the main app drawer and then choose Workspaces.

2. Click the **Add workspace** button next to the organisation name.

3. Choose a name for the workspace and click **Next**.

## Clear a workspace

Clearing a workspace removes all the nodes and edges associated with it.

> This action cannot be undone.

1. Open the profile menu from the main app drawer and then choose Settings.

2. Select Workspace Settings.

3. Click the **Clear workspace** button.
