# Setup Emails

If appropriately configured, Grai can send emails for a variety of purposes including reseting passwords and inviting new users.

<Callout type="info" emoji="ℹ️">
  Emails are setup for you with Grai cloud, making it easy to get started.
</Callout>


## Default Configuration

The Web App depends on the Django framework and works with a variety of email backends.
You can find additional information about these configuration settings in the core Django documentation, see [https://docs.djangoproject.com/en/4.1/topics/email/](https://docs.djangoproject.com/en/4.1/topics/email/).

You will need to set the following environment variables:

| Variable      | Value                                          |
| ------------- | ---------------------------------------------- |
| EMAIL_BACKEND | django.core.mail.backends.console.EmailBackend |
| EMAIL_FROM    | Email address your emails will arrive from     |

## AWS SES Email

If you would like to use AWS's SES service to provision your emails you'll need to configure a few
additional environment variables.

<Callout type="info" emoji="ℹ️">
  See [https://aws.amazon.com/ses/](https://aws.amazon.com/ses/) for more details.
</Callout>


| Variable                | Value                                          |
| ----------------------- | ---------------------------------------------- |
| AWS_ACCESS_KEY_ID       | Your AWS access key id                         |
| AWS_SECRET_ACCESS_KEY   | Your AWS secret access key                     |
| AWS_SES_REGION_NAME     | The AWS region for SES                         |
| AWS_SES_REGION_ENDPOINT | The AWS SES region endpoint                    |
