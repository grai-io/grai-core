# Grai Web App

## Introduction

The Web App is the primary GUI component of Grai.
With it you can create and manage most components of Grai including adding new integrations, viewing your data dependency graph, creating new data sources, and more.

## Dependencies

The frontend is a [React](https://reactjs.org/) application written in [Typescript](https://www.typescriptlang.org/).
It communicates with the backend server through the [GraphQL API](/server/graphql-api).
