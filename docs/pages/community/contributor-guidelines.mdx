# Contributor Guidelines

Grai is an open-source and self-hosted Data Lineage Platform and we love developer contributions!
This section covers basic instructions for developing new features in Grai.

If you're unsure about how to start contributing or have any questions even after reading them join our [slack](https://join.slack.com/t/graicommunity/shared_invite/zt-1il70kfeb-TaCm5fwHg_quWCpKNYyj6w).
If you're looking for a good place to start check out one of the open [issues](https://github.com/grai-io/grai-core/issues) on github.

<Callout type="warning" emoji="ℹ️">
  By contributing to Grai, you are deemed to have accepted the [Developer
  Certificate of Origin](https://developercertificate.org/) and agreed to
  GitHub's [Community
  Guidelines](https://docs.github.com/en/site-policy/github-terms/github-community-guidelines)
  and [Acceptable Use
  Policy](https://docs.github.com/en/site-policy/acceptable-use-policies/github-acceptable-use-policies)
</Callout>

**If you're ready to get started, you should first clone our GitHub [repository](https://github.com/grai-io/grai-core)**

## Reporting Bugs

To report bugs, please create an issue on the project [here](https://github.com/grai-io/grai-core/issues).

## Reporting Security Issues

Please do not create a public GitHub issue for a security issue, contact us at support@grai.io.

## Requesting Features

To request new features, please create an issue on the project [here](https://github.com/grai-io/grai-core/issues).

## What about Integrations?

To request a new integration, raise a GitHub issue or ask in our slack channel. We may already be working on one!

To contribute an integration yourself, guidelines on how to write a new Grai integration coming soon!

## Where can I Go for Help?

If you need help, come and join our [slack](https://join.slack.com/t/graicommunity/shared_invite/zt-1il70kfeb-TaCm5fwHg_quWCpKNYyj6w).

## What does the Code of Conduct mean for me?

Our Code of Conduct means that you are responsible for treating everyone on the project with respect and courtesy regardless of their identity. If you are the victim of any inappropriate behavior or comments as described in our Code of Conduct, we are here for you and will do the best to ensure that the abuser is reprimanded appropriately, per our code.
